UB ImGui

UB ImGui é uma implementação, wrapper e binds do Dear ImGui para unity.
Ele te permite criar interface inteiramente por código de forma simple e intuitiva, usando uma api de modo imediato.

Tecnologias

  • Unity
  • C#
  • C++

Implementação

Primeiro foi criado o SharpImgui, um gerador de binds de c++ para c#.
Foi feito ajustes na api para ser compativel com a unity (abstrair funções inseguras, criar delegates para eventos do c++).
E a parte mais desafiadora que foi criar um backend da unity, converter texturas da engine em texturas nativas, renderizar na tela do jogo, mandar inputs da engine para a api, ciclo de vida correto.
