UB ImGui

UB ImGui é uma implementação, wrapper e binds do Dear ImGui para unity, permitindo a criação de interfaces inteiramente por código utilizando uma API de modo imediato.

Tecnologias

  • Unity
  • C#
  • C++

Implementação

Primeiro foi criado o SharpImgui, um gerador de binds de c++ para c#.
A API foi adaptada para o ambiente da unity abstraindo funcionalidades inseguras e garantindo compatibilidade com o ciclo de vida da engine.
O maior desafio foi a criação de um backend específico para Unity, incluindo conversão de texturas da engine para o formato nativo, renderização na tela do jogo, redirecionamento de inputs e interoperabilidade entre C++ e C#.
