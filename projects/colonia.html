Colonia

Projeto Colónia é um jogo plataforma 2d de grande escala desenvolvido em equipe na Unity. Atuei principalmente na evolução arquitetónica do projeto, criação de sistemas centrais de gameplay e refatoração/correções de bugs.

Tecnologias

  • Unity
  • C#

Meu papel

  • Arquitetura de sistemas
  • Criação de ferramentas internas
  • Correções de bugs e suporte estrutural
  • Refatoração progressiva

Destaques

  • Sistema de Controle

    Um sistema de controle desacoplado da origem, permitindo trocar dinamicamente a fonte (input do jogador, AI, scripts, cutscenes) sem alterar o código consumidor.
    O mesmo sistema é utilizando tanto para o player quanto para os inimigos e chefes, facilitando automação, cutscenes e testes.

  • Cutscenes e Soft Cutscenes

    Criação de diversas cutscenes e soft cutscenes utilizando o sistema de controle unificado, permitindo transições suaves entre sessões de gameplay e controle scriptado.

  • FSM + GOAP

    Sistema de FSM baseado em GOAP, para o player, inimigos e bosses.
    Foi arquitetado para ser reutilizável, extensível e modular, permitindo a criação de novos comportamentos e a reutilização deles de forma simples.
    Inclui ferramentas de editor para configuração e debugging dos comportamentos.
