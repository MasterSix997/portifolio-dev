Drawbug

Drawbug é um renderizador de formas de alta porformance, com o foco principal em debug

Tecnologias

  • Unity
  • C#

Implementação

Possui diversas formas 2D e 3D criadas por código (Cubo, Esfera, Capsula, Cilindro, Setas, ...)
Cada forma possui uma variante solida ou linha.
Todas as chamadas da api são compativeis com o burst, e guardadas em um buffer que é processado em paralelo no final do frame e exibido na tela de forma imediata (1 drawcall).

Pra isso for preciso usar o sistema de Render Graph da unity, uma api mais baixo nivel que fornece metodos para interagir com a renderização interna da engine, jobs system para processamento em paralelo, e unity collections para criar buffers especializados
