Drawbug

Drawbug é um renderizador de formas de alta performance para Unity, com foco em debug visual e integração com o pipeline de renderização da engine.

Tecnologias

  • Unity
  • C#

Detalhes Técnicos

  • Integração direta com Render Graph
  • Suporte a URP, HDRP e Built-in
  • Arquitetura orientada a API, evitando padrões monolíticos
  • Uso de Jobs e Burst para geração de dados
  • Editor customizado para configuração

Implementação

Possui diversas formas 2D e 3D geradas por código (Cubo, Esfera, Cápsula, Cilindro, Setas, etc), com variação entre modo sólido ou linha.
As chamadas de API são compatíveis com Burst e armazenadas em buffers especializados, processados em paralelo ao final do frame e renderizados em um único draw call.

A implementação exigiu o uso do Render Graph da Unity, uma API de baixo nível para integração com o sistema interno de renderização, Job System e Unity Collections para gerenciamento eficiente de memória e paralelismo.
